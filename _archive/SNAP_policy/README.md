# SNAP policy (Research Materials)

This directory is a research-material workspace (drafts, literature, proposal assets, and historical extracts).

## Status in pipeline
- Not a canonical input for `R/99_run/run_all.R`.
- Files here are references/manual materials unless explicitly imported by scripts.

## Subfolders
- `Literature Review/` — papers and notes
- `prediction/` — exploratory prediction work
- `FAP_output/` — historical extracted CSV snapshots
- `previous proposal/`, `soda/`, `food retailer/`, `online platform/` — topic-specific drafts
- `_archive_temp/` — temporary lock files moved from root (`~$*.docx`)

## Rule
If a file becomes a true pipeline input, copy it into canonical data paths (`data/raw`, `data/derived`, or `data/external`) and document usage in root `README.md`.
